# 📁 /docs – System Continuity + Handoff Layer

This folder contains all source-of-truth documentation for **structure**, **philosophy**, and **handoff continuity** across development sessions.

If you're joining this project midstream, **read these in order**:

1. `GOALS.md` – immutable principles and constraints
2. `ARCHITECTURE.md` – source layout, casing, modular rules
3. `TEMPLATING.md` – slot/layout system and rendering engine
4. `SAVEPOINT.md` – enforced rules that must not break
5. `ROADMAP.md` – active checklist from now to launch
6. `HANDOFF.md` – session-by-session record of progress

Each doc is both:
- **Human readable**
- **Machine-traversable** via headings, checkboxes, and fenced code blocks

> Update `ROADMAP.md` and `HANDOFF.md` at the end of every work session.
